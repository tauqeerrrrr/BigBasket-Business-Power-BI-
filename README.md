# BigBasket-Business-Power-BI-

# 🛒 BigBasket Business Analytics Project

A complete **Data Analytics Project** using **Python, SQL, and Power BI** to analyze BigBasket product data and generate business insights.

---

# 📌 Project Overview

This project demonstrates an end-to-end analytics workflow:

- Data Cleaning using Python
- SQL Business Analysis
- Power BI Dashboard Development
- Business Insight Generation
- KPI Reporting

The objective is to understand product performance, customer pricing behavior, discount strategy, and brand performance using real-world retail data.
